GitHub Sync Setup

Step 1: Go to github.com/settings/tokens/new

Step 2: Under "Note", type: Bloodink Sync

Step 3: Under "Select scopes", check ONLY: gist

Step 4: Click "Generate token" at bottom

Step 5: Copy the token (starts with ghp_)

Step 6: Paste it in Settings → GitHub Token

⚠️ Save the token somewhere safe - GitHub only shows it once!
