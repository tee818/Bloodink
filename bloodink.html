GitHub Sync Setup

Step 1: Go to github.com/settings/tokens/new

Step 2: Note: Bloodink Sync — scope: gist only

Step 3: Generate, copy the token (ghp_...), paste into Settings → GitHub Token → Save

Step 4: Hit Sync from the sidebar once. A gist will be created.

Step 5: Go to gist.github.com, find the Bloodink gist, copy its ID from the URL

Step 6: Paste that ID into Settings → Gist ID on both devices → Save

⚠️ Save the token somewhere safe — GitHub only shows it once!
